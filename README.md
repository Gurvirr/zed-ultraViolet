## Usage
1. Open **Zed**
2. Navigate to the settings menu ( <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>X</kbd> )
3. Search for '**ultraViolet**' & click install
4. Select '**ultraViolet**' in the theme selector ( <kbd>Ctrl</kbd>+<kbd>K</kbd>+<kbd>T</kbd> )
5. Enjoy!
